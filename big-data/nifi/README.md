# Nifi

## Directories

```console
/usr/local/Cellar/nifi/1.11.4/libexec/logs
/usr/local/Cellar/nifi/1.11.4/libexec/conf
/usr/local/Cellar/nifi/1.11.4/libexec
```

## Start foreground

```console
nifi run
```

## Start background

```console
nifi-start
```

## Stop

```console
nifi-stop
```

## Status

```console
nifi-status
```

## Validate in browser

http://localhost:8080/nifi/
